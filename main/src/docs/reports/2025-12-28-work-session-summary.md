## 2. 技术实现

### 2.1 权限标识规范

采用 `模块:功能:操作` 的三段式命名：

```plain
system:user:add      # 系统管理 - 用户 - 新增
base:goods:edit      # 基础数据 - 商品 - 编辑
inventory:check:add  # 库存管理 - 盘点 - 新增
```

### 2.2 使用方式

```vue
<template>
	<!-- 工具栏按钮 -->
	<el-button v-perms="['system:user:add']" type="primary" @click="handleAdd"> 新增用户 </el-button>

	<!-- 表格操作按钮 -->
	<el-button v-perms="['system:user:edit']" type="primary" link @click="handleEdit(row)"> 编辑 </el-button>
</template>
```

### 2.3 权限检查流程

1. 用户登录后获取菜单数据（包含按钮权限）
2. 提取所有 `type === '2'` 的菜单项的 `permission` 字段
3. 存储到用户 store 的 `permissions` 数组
4. `v-perms` 指令检查用户权限，无权限则移除按钮
