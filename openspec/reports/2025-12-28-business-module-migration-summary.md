### 3.3 代码规范

**文件结构：**

```vue
<template>
	<!-- 页面结构 -->
</template>

<script setup lang="ts">
// 导入
// 类型定义
// 响应式数据
// 方法定义
// 生命周期
</script>

<style scoped lang="scss">
// 样式
</style>
```

**命名规范：**

- 组件名：PascalCase（如 `BaseDataGoods`）
- 变量名：camelCase（如 `tableData`）
- 常量名：UPPER_CASE（如 `API_URL`）
- 文件名：kebab-case（如 `goods-management.vue`）
