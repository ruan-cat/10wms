## 5. 技术亮点

### 5.1 主题变量适配

使用 CSS 变量实现主题自动适配：

```scss
.dynamic-form {
	// 使用 Pure-Admin 的主题变量
	--bc: var(--el-border-color);

	&.table-like-bg {
		.el-form-item__label {
			color: var(--el-text-color-regular);
			background-color: var(--el-fill-color-light);
		}
	}
}
```

### 5.2 泛型类型支持
