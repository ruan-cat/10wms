## 6. 迁移方法总结

### 6.1 迁移步骤

1. **分析 Origin API**
   - 查看 Origin 项目的 API 文件
   - 理解接口功能和参数
   - 记录特殊处理逻辑

2. **定义类型**
   - 定义请求参数类型
   - 定义响应数据类型
   - 使用 TypeScript 接口

3. **转换 API 调用**
   - 从 Request 类转换为 http 工具类
   - 从回调方式转换为 Promise 方式
   - 统一错误处理

4. **添加注释**
   - 使用 JSDoc 格式
   - 说明接口功能
   - 标注参数说明

### 6.2 转换示例

**Origin 项目（回调方式）：**

```javascript
export async function getshangpingApi(data, success, fail) {
	try {
		const res = await Request.requestForm(Request.GET, "/md/goods/query-all", data);
		if (res.data) {
			success(res.data);
		}
	} catch (err) {
		console.warn(err);
		fail();
	}
}
```

**Pure-Admin 项目（Promise 方式）：**

```typescript
/**
 * 获取商品列表（分页）
 */
export const getGoodsList = (params: GoodsQueryParams) => {
	return http.request<PageResult<GoodsInfo>>("get", "/md/goods/query-all", { params });
};
```
