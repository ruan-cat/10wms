## 9. 重要决策

### 9.1 架构决策

|  决策点  |         方案          |               理由                |
| :------: | :-------------------: | :-------------------------------: |
| HTTP 层  |   CustomHttp 类封装   |    保留 Origin 逻辑，兼容性好     |
| 状态管理 | 扩展 Pure-Admin store |   兼容 Origin API，复用现有功能   |
| 路由系统 |      统一元信息       |     集成守卫逻辑，保持一致性      |
| 表格组件 |      封装适配层       | 兼容 Origin API + Pure-Admin 功能 |

### 9.2 技术选型

| 组件类型 |       选型       |        说明        |
| :------: | :--------------: | :----------------: |
|   表格   | @pureadmin/table | 功能强大，易于扩展 |
|   表单   | Origin base-form |    保留原有逻辑    |
|   分页   |   Element Plus   | 原生组件，无需封装 |
|  对话框  |   Element Plus   | 原生组件，功能完善 |
